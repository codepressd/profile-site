Eddie Out
: Github Repo

Technology used: Node.js Application

  • Javascript
  • JQuery
  • Mongodb
  • Node.js

Project Description:

Some of the best times of my life have been running rivers all over the western United States. For many seasons I was a whitewater river guide and more recently I've been getting in to whitewater kayaking. There are a few sites where you can grab river flow information, but the majority are hard to navigate.

I created Eddie Out with the hope of getting up to date river flow information easier and more visually appealing. This site still has a long way to go, and I hope to add features where fellow boaters can connect and run trips together. In the near future I plan to convert this site into a full stack application for better UX and more robust features.

Salut.io
: Github Repo

Technology used: Full Stack App

  • Mongodb
  • Express
  • Node.js
  • React

Project Description:

I've worked in Restaurants for over 15 years and managed them for almost a decade. One thing that is always a pain in the rear, is trying to keep track of ordering and doing it efficiently.

With the current restaurant I run, we have well over 20 different suppliers and sometime multiple different reps for each supplier. Also each one of those suppliers has sometimes 1000's of products, so I wanted to create a platform that makes ordering a lot easier for restaurants. This app still has a long way to go, but the initial bones are here and I'm excited to start adding new features
